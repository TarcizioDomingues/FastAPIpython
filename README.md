# API de Compras - FastAPI

Projeto CRUD de compras usando:

- FastAPI
- SQLModel
- SQLite
- Uvicorn

## Funcionalidades

- Criar compras
- Listar compras
- Atualizar compras
- Deletar compras

## Instalação

```bash
pip install -r requirements.txt
```

## Rodar projeto

```bash
uvicorn main:app --reload
```

## Documentação

```text
http://127.0.0.1:8000/docs
```
